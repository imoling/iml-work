# 标讯查询标准作业流程

## 1. 意图确认与关键词提取
- 接收用户输入，识别其包含的核心实体：
  - **关键词**：如行业术语、项目名称、技术领域等。
  - **客户/企业名**：明确需要查询的特定主体。
- 若用户未提供明确关键词或企业名称，主动询问：“请问您想查询哪个行业或哪家公司的标讯？”

## 2. 搜索策略构建
- 基于提取的信息，构建搜索引擎查询语句：
  - 优先组合：`[企业名称/关键词] + [招标/中标/公告]`
  - 若指定时间范围，加入时间限定词，如 `2023年` 或 `最近一个月`。
- 确定目标数据源范围：中国招标投标公共服务平台、各地政府采购网、大型商业招标聚合站。

## 3. 执行 Playwright 自动化抓取
- 启动 Playwright 浏览器实例，设置适当的请求头和延迟以避免被反爬机制拦截。
- 访问主流标讯聚合平台或搜索引擎结果页。
- 输入构建好的搜索语句并执行搜索。

## 4. 数据清洗与筛选
- 解析搜索结果页，提取标题、发布时间、发布单位、链接URL等关键字段。
- **去重处理**：过滤掉内容重复或无效的搜索结果。
- **相关性排序**：根据关键词匹配度、发布时间新旧进行排序，优先展示最新且高度相关的信息。
- 若搜索结果不足，尝试扩大关键词范围或替换为同义词重新搜索一次。

## 5. 结果反馈
- 将筛选后的前 5-10 条最高相关标讯整理为结构化列表。
- 输出格式示例：
  1. **[发布时间]** 标题概要... [详情链接]
  2. **[发布时间]** 标题概要... [详情链接]
- 若未找到相关信息，回复：“未查询到与【[关键词]】相关的最新标讯，建议您尝试更换关键词或检查企业名称是否准确。”
- 询问用户是否需要查看某条详情的具体链接或进一步筛选（如按地区、金额筛选）。
