# 查看个人待办工作 SOP

## 执行步骤
1. 打开绑定的业务系统网页，地址来自业务系统连接，登录会话由客户端自动注入，无需输入账号密码。
2. 在系统首页或导航栏中，点击 **“统一待办”** 菜单项，进入个人待办工作列表页面。
3. 等待系统加载并展示当前登录用户所有未完成的待办事项列表，包括各事项的标题、提交人、提交时间、紧急程度等字段信息。
4. 自动抓取列表中的所有待办事项，并按系统默认排序（通常按提交时间倒序）整理为结构化数据。

## 反馈要求
- **成功反馈**：将抓取到的待办事项逐条展示给用户，包括每条待办的标题、提交人、提交时间、紧急程度等信息。示例格式如下：
  - 标题：【紧急】2024年Q3预算审批
  - 提交人：张三
  - 提交时间：2024-08-15 14:30
  - 紧急程度：高

- **结果为空时提示**：若列表为空，向用户反馈“当前没有待办事项，请稍后再查”。

- **列表过长时处理**：若待办事项超过20条，仅展示前20条，并在末尾补充提示“共有N条待办，此处仅显示前20条”，其中N为实际总数。

- **异常处理**：
  - 若系统要求重新登录或会话过期，向用户反馈“登录会话已失效，请重新登录后重试”。
  - 若当前用户无权限查看待办，向用户反馈“您暂无查看待办的权限，请联系管理员”。
  - 若页面弹窗、广告或拦截提醒影响操作，向用户反馈“页面存在干扰元素，请手动关闭弹窗后重新执行”。
