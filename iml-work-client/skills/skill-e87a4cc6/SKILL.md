# OA统一待办获取 SOP

## 执行步骤
1. 打开绑定的OA业务系统（地址来自业务系统连接，登录会话由客户端注入，无需输入账号密码）。
2. 在首页导航中定位并进入「统一待办」入口。
3. 等待待办列表加载完成，抓取每条待办的标题、类型与发起人。
4. 将待办整理为清单返回，并向用户汇总：共有多少条、按类型分布、是否有紧急项。

## 反馈要求
- 若待办为空，明确告知「当前没有待处理事项」。
- 列表过长时只展示前若干条，并提示总数。
