# iML Work 实操录制工具（独立版）

一个轻量的浏览器实操录制工具：在**本地**登录目标业务系统（CRM/OA/…）正常操作一遍，工具会捕获每一步的点击/输入与**稳健选择器**，生成可回放的技能并上传回**管理端「技能中心」**。

## 设计要点
- **登录态只在本地、且只是一次性**：录制时你自己登一下目标系统把页面点出来，工具**只上传步骤+选择器，绝不保存任何登录态/Cookie**。
- **执行在主客户端**：员工使用技能时，由 iML Work 主客户端用其本地「企业系统连接」里已登录的会话确定性回放这些步骤。
- 与主客户端内置录制器共用同一套选择器捕获逻辑（`#id > data-* > name > aria-label > nth-of-type CSS path`）。

## 使用
```bash
npm install
npm start
```
1. 填写**管理端地址**（默认 `http://localhost:8080`），点「刷新系统列表」。
2. 选择目标业务系统、填技能名称与触发关键词，点「开始录制」。
3. 在弹出的浏览器窗口里登录并正常操作一遍，完成后点「结束录制」。
4. 核对步骤，把需要每次让用户确认填写的「输入/选择」步骤勾为**可填字段**并命名。
5. 点「保存并上传技能」——技能即进入管理端技能中心，可被各岗位分身按权限调用。
